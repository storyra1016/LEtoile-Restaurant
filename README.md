# L’Étoile Restaurant

## 法式餐廳智慧會員點餐推薦系統

本專題以高級法式餐廳為情境，使用 **C# Windows Forms** 與 **SQL Server** 開發一套模擬餐廳 POS 流程的會員點餐系統。系統整合會員登入、會員註冊、餐點選購、推薦搭配、購物車結帳與熱銷分析功能，訓練我在程式設計、資料庫應用、事件驅動邏輯、資料視覺化與系統整合上的能力。

## 四、功能詳細說明

### 1. SQL 會員資料建置

系統使用 SQL Server 建立 Members 會員資料表，用於儲存會員基本資料與會員狀態。

資料表主要包含：

* 會員姓名
* 電話號碼
* Email
* VIP 狀態
* 折扣率
* 註冊日期
* 累積消費金額

這些資料會作為後續會員登入驗證、VIP 判斷與結帳折扣計算的依據。

---

### 2. 會員登入驗證

使用者可透過會員電話號碼登入系統。系統會依照輸入資料查詢 SQL Server 中的 Members 資料表，判斷使用者是否為已註冊會員。

登入後，系統會根據會員狀態判斷：

* 一般會員
* VIP 會員

會員狀態會影響後續購物車結帳時的折扣計算。

---

### 3. 新會員註冊

系統提供會員註冊功能，使用者可輸入：

* 姓名
* 電話
* Email
* 是否為 VIP 會員

註冊完成後，會員資料可作為後續登入驗證、會員資料管理與折扣判斷的基礎。

---

### 4. 餐點分類選購

餐點依照類別進行分類，讓使用者能更清楚選擇餐點。

目前系統包含的餐點分類有：

* 前菜
* 主餐
* 甜點
* 飲品

使用者點選餐點後，系統會同步顯示餐點名稱、價格、介紹與圖片，模擬實際餐廳點餐介面。

---

### 5. 餐點圖片與價格顯示

系統使用 PictureBox 顯示餐點圖片，並搭配文字區塊顯示餐點資訊。

此功能的目的是讓使用者在點餐時能同時看到：

* 餐點圖片
* 餐點價格
* 餐點分類
* 餐點特色介紹

這讓系統不只是單純清單選取，而是更接近實際餐廳點餐系統的操作體驗。

---

### 6. 推薦系統

本系統使用規則式推薦邏輯。當使用者選擇主餐後，系統會依照主餐特色推薦適合搭配的飲品與甜點。

例如：
- 紅酒燉牛肉 → 推薦搭配紅酒與焦糖布蕾
- 法式羊排 → 推薦搭配紅酒與馬卡龍
- 香煎鴨胸 → 推薦搭配紅酒與馬卡龍

推薦功能會顯示推薦結果與推薦原因，提升點餐互動性，也訓練我設計條件判斷與規則邏輯的能力。

---

### 7. 即時點餐摘要

在餐點選購過程中，系統會即時顯示目前已加入的餐點內容，讓使用者在進入購物車前能先確認目前點餐狀態。

此功能可避免使用者重複點餐或忘記已加入的品項，提升系統操作的清楚度。

---

### 8. 購物車結帳

購物車頁使用 DataGridView 顯示已選餐點資料，包含：

* 餐點名稱
* 單價
* 數量
* 小計

結帳時，系統會依會員狀態自動計算折扣與實付金額，模擬餐廳 POS 結帳流程。

此功能訓練我處理：

* DataGridView 資料顯示
* 數量與小計計算
* 會員折扣套用
* 結帳提示訊息
* 購物車資料更新

---

### 9. 熱銷分析圖表

系統使用 Chart 圖表呈現餐點銷售資料，包含長條圖與圓餅圖。

熱銷分析功能可顯示：

* 熱門餐點排行
* 餐點銷售比例
* 不同餐點之間的銷售差異

此功能讓我練習將銷售資料轉換成視覺化圖表，也提升我對資料分析與資料呈現的理解。

## 六、資料庫建置說明

本專題使用 SQL Server 建立 RestaurantDB 資料庫，並透過 CreateTable.sql 建立 Members 會員資料表。

執行系統前，需先確認：

1. 已安裝 SQL Server
2. 已建立 RestaurantDB
3. 已執行 CreateTable.sql
4. App.config 連線字串已設定正確
5. Visual Studio 可正常連接資料庫

---

## 七、執行方式

1. 下載或 Clone 本專案
2. 使用 Visual Studio 2022 開啟專案
3. 於 SQL Server 建立 RestaurantDB
4. 執行 CreateTable.sql 建立 Members 資料表
5. 確認 App.config 連線字串
6. 執行 Windows Forms 專案
7. 進行會員登入、餐點選購、購物車結帳與熱銷分析測試
